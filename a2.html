Goal/s

Briefly and clearly describe the measurable goal/s of your website here.

Our measurable goals for this website were:

  • Create a clear contact option

    • Designed the website to showcase multiple options on each page to allow visitors to the site to be able to get in contact with TTCMA easily.

  • Allow users to sign up and become members

    • Implemented a Login/Registration section the includes an option in the navigation bar that is clear and accessible. This will allow members to join the TTCMA community.

  • Drive the audience to the events page to stay up to date on current and upcoming events

    • Designed the events page to display current events that are created through a events manager plugin that will help prevent old events from displaying on the site. Keeping it all up to date for site visitors.
